#ETCH A SKETCH ODIN PROJECT

Basic Etch-a-sketch project. Drawing happens when user hovers over the sketchboard

## Functionalities
Variety of colors to choose from
Random color
Resize sketchboard
Erase function
Reset function

## Stack
1. HTML
2. CSS
3. JavaScript

1. DOM manipulation
2. Event handling
3. Dynamic styling
4. Working with loops
5. UI layout
6. Debugging
